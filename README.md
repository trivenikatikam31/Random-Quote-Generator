# Random Quote Generator

## Overview

A web application that displays random inspirational quotes and their authors. Users can generate a new quote by clicking a button.

## Features

* Random quote on page load
* New Quote button
* Displays quote text and author
* Clean and minimal UI
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Open the project.
2. Launch the application.
3. View the random quote.
4. Click "New Quote" to generate another quote.
